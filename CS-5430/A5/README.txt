INSTRUCTIONS
Install java 8 by running

sudo apt-get update
sudo apt-get install openjdk-8-jdk

Then change your permissions to make analyze executable by running

chmod +x analyze

Finally run the program by running

./analyze /path/to/log/file

This will produce a line of output suitable to append to syslog
